# 面別ランキング画面リデザイン — 設計書

- 日付: 2026-07-18
- 対象: Hover Attack Web / 面別ランキング表示（`stage_ranking_display`）
- 目的: (1) レイアウトと文字サイズ・配置を整える、(2) ステージ色の視認性を上げる、
  (3) 各ステージの「1カット・シーン」を描いて、面別記録であることをひと目で伝える。

## レイアウト（上シーン＋左右リスト）

```
              S T A G E  n            ← 大見出し・明るいアクセント色（中央）
      THIS WEEK · TOP 5               ← 小サブタイトル
 ┌───────── 洞窟シーン（全幅・~150px）─────────┐
 │  自機(左,バズーカ) →→ ✦ミサイル✦ ←← 敵(右)   │  ← そのステージ初登場の敵
 └────────────────────────────────────────┘
   FASTEST TIME          │        HIGH SCORE     ← 左右2カラム見出し
 1. AAA 🇯🇵 00:42.15      │   1. AAA 🇯🇵   12,500
 …5位まで                 │   …5位まで
              PRESS ANY KEY TO START
```

- 800×600 canvas。おおよそ: 見出し y≈42、サブ y≈66、シーン y≈84–232（全幅, 左右マージン40）、
  リスト見出し y≈258、行 y≈288 から `lineH≈30` で5行、フッタ y≈約H-18。
- 左カラム中心 x≈canvas.width×0.27、右カラム中心 x≈canvas.width×0.73（行は各カラム内で左寄せ）。

## 視認性（色）

- **アクセント色** = `lerpColor(palette.fill, '#FFFFFF', 0.55)`（ステージ色を明るく寄せる）。
  見出し「STAGE n」・カラム見出し・順位番号・数値に使用。
- **本文** = オフホワイト `#EAEAEA`（名前など）。国旗は絵文字そのまま。
- 生の `palette.fill` は**シーンの色調・アクセントバー・背景の縁**に使い、ステージ識別を保つ。
- 背景は暗色（`#0a0a0a`）だが、テキストはアクセント/オフホワイトで高コントラストにする。

## シーン（`src/js/ui/StageScene.js`・新規）

- 関数 `drawStageScene(ctx, x, y, w, h, stageIndex, palette, nowMs)`。
- 洞窟断面: 角丸の枠、`palette` で床/壁を淡く着色、簡単な岩・鍾乳石のシルエット。
- 左に自機（ホバー機＋バズーカ, 右向き, プレイヤー色 `#55CCDD` 系）、
  右にそのステージの**初登場の敵**（下表）を左向きで配置。中央でミサイルが交差。
- **敵マッピング**（stageIndex 0..6）:
  1. 戦車 / 2. アタッカー(standard) / 3. 砲台 / 4. ドローン /
  5. ライバル / 6. アーティラリー / 7. **巡航ミサイル**（敵本体がミサイル。自機と対峙）。
- スプライトは**専用のコンパクト描画**（既存エンティティ`draw()`は`game`結合が強く縮尺も合わないため流用しない）。
  敵はそれぞれの代表色で塗り分け（アタッカー系は `ENEMY_ATTACKER_TYPES` の bodyColor 等を参考に簡略化）。
- **軽アニメ**: `nowMs` を位相に、ミサイルが左右を往復（発射→交差→着弾のループ）、
  発射口のマズルフラッシュ、機体の微小ホバー上下。負荷は軽微（数個の図形/フレーム）。

## 到達ゲート（未到達ステージは非表示）

- **ローカルで一度も到達していないステージは巡回表示に出さない**（そのステージの敵＝シーンを見せないことで、
  初見のわくわくを残す）。
- `hoverattack_max_stage_reached`（localStorage・**週替わりではなく永続**）に、到達した最大ステージ番号を保持。
  ステージ開始時（`resetLevel`/プレイ開始）に `max(既存, missionsCompleted+1)` で更新。
- 巡回（`_updateStageRankingDisplay`）は **STAGE 1..maxReached** のみを順に表示。`maxReached < 1`（未プレイ）なら
  `stage_ranking_display` に入らず `wall_of_fame_display` へ直行。
- 到達済みなら記録が空（トップ5未登録）でもそのステージは表示する（＝敵は既知なので隠す必要なし）。
